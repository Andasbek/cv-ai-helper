## Обзор проекта

**CV AI Helper** — веб-приложение для автоматического анализа резюме, сопоставления с вакансиями и проведения тренировочных интервью с помощью искусственного интеллекта.

### Целевая аудитория

- Студенты и junior-специалисты, готовящиеся к первому трудоустройству
- Соискатели, желающие улучшить резюме перед откликом на конкретную вакансию
- Кандидаты, которым нужна практика прохождения технических и поведенческих интервью

### Основные возможности

| Модуль | Описание |
|--------|----------|
| **A — Загрузка CV** | Загрузка PDF/TXT, извлечение текста, сегментация на секции |
| **B — Анализ CV** | AI-анализ: проблемы, советы, переформулировки |
| **C — JD Matching** | Сравнение резюме с вакансией, match score, пробелы в навыках |
| **D — Интервью** | Чат-интервью, обратная связь по каждому ответу, итоговый отчёт |

### Стек технологий

| Слой | Технология | Версия |
|------|-----------|--------|
| Frontend | Streamlit | 1.39.0 |
| Backend API | FastAPI | 0.115.0 |
| ASGI-сервер | Uvicorn | 0.32.0 |
| ORM | SQLAlchemy (async) | 2.0.36 |
| База данных | SQLite + aiosqlite | — |
| AI-провайдер | OpenAI GPT-4o | SDK 1.54.0 |
| Парсер PDF | pdfplumber | 0.11.4 |
| Валидация данных | Pydantic v2 | 2.9.2 |

│   ├── models/

│   │   └── interview.py      # Эндпоинты /api/interview/*
│   ├── services/
│   │   ├── parser.py         # Извлечение текста из файлов

│   │   ├── cv_analyzer.py    # AI-анализ резюме
